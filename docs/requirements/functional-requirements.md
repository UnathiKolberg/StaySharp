# Functional Requirements

## User Management

### Features
Users can:
- Register an account
- Login
- Logout
- Reset password

### System Requirements
The system shall:
- Validate email uniqueness
- Encrypt passwords
- Authenticate users

---

## Quiz Engine

### Features
Users can:
- Start quizzes
- Skip questions
- Review answers
- Submit quizzes

### System Requirements
The system shall:
- Randomize questions
- Randomize options
- Calculate scores
- Store results
