# StaySharp
StaySharp is a web application designed to help Computer Science graduates maintain and improve their technical skills through quizzes, coding exercises, interview preparation, and gamification.

## Features

- C# Quizzes
- Java Quizzes
- SQL Exercises
- .NET Questions
- Interview Preparation
- XP & Leveling System
- Achievement Badges
- Daily Challenges
- Progress Tracking

## Technology Stack

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQLite
- ASP.NET Identity
- Bootstrap
